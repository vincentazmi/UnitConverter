# Unit Converter!
# Script that can convert the following units into their equivalents:
1. Weight
2. Distance
3. Speed
4. Volume
5. Temperature

e.g Weight:
1) Kilograms
2) Grams
3) Milligrams    
4) Ounce
5) Pound
6) Stone
7) Tonne

Input: 55KG
Gets you the following output:

55.0 kg
55000.0 g
55000000.0 mg
1940.07 oz
121.25 lb
8.661 st
0.055 t
